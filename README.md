# דירוג דירות — פרויקט HAKEREM, כרמי גת צפון

כלי החלטה אינטראקטיבי לבחירת דירה במסגרת "מחיר מטרה".
דירוג של 58 דירות 3.5 חדרים (מהמועדפת ביותר לפחות מועדפת) + גיבוי 4 חדרים,
עם יתרונות/חסרונות לכל דירה, לפי כיוון שמש, דגם, קומה ומחיר.

## שימוש

- **דשבורד אינטראקטיבי (לנייד):** [index.html](https://amitrubin10.github.io/DIRA/) — חיפוש לפי בניין/דירה, סימון דירות שנלקחו, סנכרון מלאי אוטומטי, וכפתור "הדירה הפנויה הבאה". ניתן להדפיס ישירות מהדפדפן.

## איך משתמשים בפגישה

כשמישהו לפניך בוחר דירה — סמן אותה "נלקחה". בכל שלב לחץ על **⭐ הדירה הפנויה הבאה שלי**
והכלי יקפיץ אותך לדירה הכי גבוהה בדירוג שעדיין פנויה.
